# Ключевые результаты для диплома

## Архитектура прогнозирования
- Базовая LSTM: обучена на ERA5 (СПб 2020–2025), заморожена навсегда
- Коррекционная модель: обучается вручную на локальных данных (~еженедельно)
- Автоматическое переобучение LSTM: УДАЛЕНО (причина: нестабильность MAE, деградация до 7°C)

## Результаты валидации (2026-05-14 — 2026-05-16, N=325 образцов по 5 мин)

| Режим              | MAE темп +1ч | MAE темп +2ч | MAE темп +3ч |
|--------------------|-------------|-------------|-------------|
| LSTM + коррекция   | 0.35°C      | 0.50°C      | 0.47°C      |
| Online API         | 0.89°C      | 0.99°C      | 1.07°C      |
| Базовый LSTM       | 1.30°C      | 1.65°C      | 1.75°C      |

| Режим              | MAE давл +1ч | MAE давл +2ч | MAE давл +3ч |
|--------------------|-------------|-------------|-------------|
| Базовый LSTM       | 0.49 hPa    | 0.62 hPa    | 0.69 hPa    |
| LSTM + коррекция   | 0.69 hPa    | 0.70 hPa    | 0.55 hPa    |
| Online API         | 9.0 hPa*    | 9.6 hPa*    | 10.0 hPa*   |

*Online API давление: систематическое смещение ~10 hPa из-за расхождения высоты рельефа
в сетке модели NWP (~130 м) и реальной высоты по GPS (~50 м).

## Вывод
Локальная LSTM-модель с коррекцией превосходит Online API в 2.5 раза по температуре.
Коррекционная модель переобучается вручную раз в 1–2 недели командой:
    python research/calibration_cli.py train-correction

## Команды управления

    # Валидация на отложенной выборке
    python research/calibration_cli.py validate

    # Статус и обучение коррекционной модели
    python research/calibration_cli.py correction-status
    python research/calibration_cli.py train-correction

    # Статус LSTM (информация, без переобучения)
    python research/calibration_cli.py lstm-status

    # Общий статус исследования
    python research/research_cli.py status
    python research/research_cli.py report
