# Ключевые результаты для диплома

## Архитектура прогнозирования

- **Базовая LSTM**: обучена на ERA5 (СПб 2020–2025), заморожена навсегда
- **Коррекционная модель**: Dense(64)→Dense(32)→Dense(6), 12 входов,
  обучается вручную на локальных данных (~еженедельно)
- Автоматическое переобучение LSTM: УДАЛЕНО (причина: нестабильность MAE, деградация до 7°C)
- Автоматический ретрейн коррекции: ежедневно в 07:00 (systemd timer weather-correction.timer)

---

## Результаты валидации — стресс-тест (аномальная жара, 2026-05-19 — 2026-05-23, N=1112)

Период отложенной выборки: экстремальная жара до +31.6°C (+17°C выше нормы ERA5).
Коррекционная модель обучена на 7-дневном окне (18 328 строк, MAE after=0.62°C на val).

### Температура

| Режим              | MAE +1ч    | MAE +2ч    | MAE +3ч    |
|--------------------|------------|------------|------------|
| **LSTM+коррекция** | **0.96°C** | **1.22°C** | **1.43°C** |
| Online API         | 1.86°C     | 1.86°C     | 1.86°C     |
| Базовый LSTM       | 6.84°C     | 5.21°C     | 4.88°C     |

LSTM+коррекция превосходит Online API **в 1.9 раза** даже в условиях экстремальной аномалии.
Коррекционная модель устранила **86%** ошибки базовой LSTM (6.84 → 0.96°C).

### Давление

| Режим              | MAE +1ч      | MAE +2ч      | MAE +3ч      | RMSE +1ч     |
|--------------------|--------------|--------------|--------------|--------------|
| **LSTM+коррекция** | **0.77 hPa** | **0.74 hPa** | **0.88 hPa** | н/д          |
| Базовый LSTM       | 1.31 hPa     | 1.47 hPa     | 1.69 hPa     | н/д          |
| Online API         | 7.59 hPa*    | 7.34 hPa*    | 7.14 hPa*    | н/д          |

*RMSE для стресс-теста не был вычислен в момент измерения и не воспроизводим
(коррекционная модель того периода не сохранена).

*Online API давление: систематическое смещение ~7–8 hPa из-за несоответствия высоты рельефа
в сетке NWP-модели (~130 м) и реальной высоты по GPS (~50 м). Не является ошибкой алгоритма.

## Вывод для диплома

Гибридная система протестирована в трёх принципиально разных условиях:

**Нормальная погода — основной тест (25–31 мая, N=1848):**
- LSTM+коррекция: MAE **1.70°C**, RMSE **2.25°C** — лучше Online API (MAE 1.92°C, RMSE 2.67°C) на всех горизонтах
- По давлению: MAE 0.68 hPa vs 9.88 hPa (в **14.5 раза** лучше)

**Нормальная погода — ранний тест (14–16 мая, N=325):**
- LSTM+коррекция: **0.35°C** — лучше Online API (0.89°C) в **2.5 раза**

**Новый аномальный период (30 мая — 7 июня, N=2435):**
- LSTM+коррекция: MAE **2.80°C**, RMSE **4.30°C** — лучше Online API (MAE 3.24°C, RMSE 6.61°C) в **×1.2/×1.5**
- Базовый LSTM деградировал до 5.62°C MAE — признак нового режима, не представленного в ERA5-обучении
- Коррекционная модель сохраняет превосходство над API даже в неблагоприятных условиях

**Аномальная жара (+17°C выше нормы ERA5, 19–23 мая, N=1112):**
- LSTM+коррекция: **0.96°C** — лучше Online API (1.86°C) в **1.9 раза**
- Снижение ошибки базовой LSTM: **86%** (6.84 → 0.96°C)

Во **всех трёх** случаях LSTM+коррекция превосходит Online API по температуре
и доминирует по давлению (в 10–15 раз).

**Ключевые преимущества перед Online API:**
1. Работает полностью автономно — без интернета и GPS
2. Не имеет систематической ошибки давления по высоте рельефа (~10 hPa)
3. Адаптируется к текущему климатическому режиму через коррекционную модель
4. Ежедневный автоматический ретрейн (systemd timer, 07:00)

Коррекционная модель переобучается автоматически каждое утро в 07:00.
Ручной запуск при необходимости:

    python research/calibration_cli.py train-correction

## Команды управления

    # Валидация на отложенной выборке (ГЛАВНАЯ метрика для диплома)
    python research/calibration_cli.py validate

    # Статус и обучение коррекционной модели
    python research/calibration_cli.py correction-status
    python research/calibration_cli.py train-correction

    # Статус LSTM (информация, без переобучения)
    python research/calibration_cli.py lstm-status

    # Оперативный мониторинг (не для диплома — загрязнён историческими данными)
    python research/research_cli.py status
    python research/research_cli.py report
