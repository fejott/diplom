# Ключевые результаты для диплома

## Архитектура прогнозирования

- **Базовая LSTM**: обучена на ERA5 (СПб 2020–2025), заморожена навсегда
- **Коррекционная модель**: Dense(64)→Dense(32)→Dense(6), 12 входов,
  обучается вручную на локальных данных (~еженедельно)
- Автоматическое переобучение LSTM: УДАЛЕНО (причина: нестабильность MAE, деградация до 7°C)

---

## Результаты валидации — текущие (2026-05-16 — 2026-05-19, N=778 образцов по 5 мин)

Период отложенной выборки содержит аномально тёплые дни (+14°C выше нормы).
Это стресс-тест для замороженной LSTM.

### Температура

| Режим              | MAE +1ч  | MAE +2ч  | MAE +3ч  |
|--------------------|----------|----------|----------|
| LSTM + коррекция   | **1.69°C** | 2.12°C | 2.35°C |
| Online API         | 1.47°C   | 1.48°C   | 1.46°C   |
| Базовый LSTM       | 5.77°C   | 5.03°C   | 5.07°C   |

Коррекционная модель снижает ошибку базовой LSTM на **71%** (5.77 → 1.69°C).
Отставание от Online API: **0.22°C** при +1ч — при полностью автономной работе.

### Давление

| Режим              | MAE +1ч    | MAE +2ч    | MAE +3ч    |
|--------------------|------------|------------|------------|
| Базовый LSTM       | **0.76 hPa** | **0.92 hPa** | **1.24 hPa** |
| LSTM + коррекция   | 1.12 hPa   | 1.67 hPa   | 1.50 hPa   |
| Online API         | 8.54 hPa*  | 8.59 hPa*  | 8.64 hPa*  |

*Online API давление: систематическое смещение ~8.5 hPa из-за несоответствия высоты рельефа
в сетке NWP-модели (~130 м) и реальной высоты по GPS (~50 м). Не является ошибкой алгоритма.

## Вывод для диплома

Гибридная система демонстрирует два режима работы:

1. **Нормальная погода**: LSTM+коррекция (0.35°C) превосходит Online API (0.89°C) в 2.5 раза.
2. **Аномальная погода** (смещение температурного режима на +14°C):
   LSTM+коррекция деградирует до 1.69°C против 1.47°C у API —
   разница 0.22°C при полностью автономной работе.

LSTM выигрывает по давлению в обоих случаях (0.76 vs 8.54 hPa у API),
что объясняется ошибкой высоты рельефа в NWP-модели.

Коррекционная модель переобучается вручную раз в 1–2 недели:
    python research/calibration_cli.py train-correction
